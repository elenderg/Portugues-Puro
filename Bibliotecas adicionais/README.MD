# README - Bibliotecas do Compilador

## Aviso Importante

As bibliotecas apresentadas neste repositório estão **em fase inicial de desenvolvimento**. Elas foram criadas para complementar o nosso compilador, mas ainda não estão prontas para uso em produção. É fundamental que os usuários tenham em mente que essas bibliotecas:

- **Precisam de mais testes**: Muitas funcionalidades ainda não foram testadas extensivamente. Ao utilizá-las, você pode encontrar bugs ou comportamentos inesperados.
- **Podem ser alteradas**: As APIs e funcionalidades podem mudar à medida que continuamos a desenvolver e melhorar as bibliotecas. Isso significa que o código que funciona hoje pode não funcionar amanhã após uma atualização.
- **Não são parte da Biblioteca Padrão**: Neste momento, essas bibliotecas não são incorporadas de forma definitiva à Biblioteca padrão do compilador e devem ser utilizadas com cautela.
