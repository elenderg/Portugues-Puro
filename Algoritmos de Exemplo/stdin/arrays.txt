Um matriz numérica é uma matriz com uns números.


Atribua {1,2,3,4,5,6,7,8,9} para uma matriz numérica denominada matriz de exemplo.

Multiplique o último elemento desta matriz de exemplo por 2.
Multiplique o elemento da posição 1 desta matriz de exemplo pelo elemento da posição 2 desta matriz de exemplo.
Multiplique o 1º elemento desta matriz de exemplo pelo 2º elemento desta matriz de exemplo.



Uma matriz é uma estrutura com
  Um número denominado tamanho,
  Uma string denominada tipo de dados,
  Uma número denominado índice atual,
  Um ponteiro denominado primeiro elemento,
  Um ponteiro denominado último elemento.
